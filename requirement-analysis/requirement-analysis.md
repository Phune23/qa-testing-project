# Requirement Analysis

## System Overview

The system is a simple E-commerce web application.

Main features:

* User Registration
* User Login
* Browse Products
* Add Product to Cart
* Checkout Order

## Functional Requirements

FR001 - User Registration
Users can create an account using email and password.

FR002 - User Login
Users can login using valid credentials.

FR003 - Browse Products
Users can view available products.

FR004 - Add to Cart
Users can add products to the shopping cart.

FR005 - Checkout
Users can place an order with shipping information.
